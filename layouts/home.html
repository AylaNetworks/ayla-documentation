Welcome to Ayla Core Content by Ayla Networks.

Created by engineers for engineers, this site contains technical guides, references, API docs, tutorials, tips, code samples, diagrams, definitions, videos, and downloads.

With the Ayla IoT Platform, you can:

  1. Connect your device to the Ayla Cloud.
  2. Create a model of your device in the cloud.
  3. Deploy a mobile or web application to monitor/control your device via the model.
  4. Integrate device data with other business systems.

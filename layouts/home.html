Welcome

Created by engineers for engineers, this Ayla Networks site contains technical guides, references, API docs, tutorials, tips, code samples, diagrams, definitions, videos, and downloads.

Submit a website or document issue.

Working with the Ayla Platform

The documents on this site explain how to connect your device to the Ayla Cloud, create a model of your device in the cloud, write/deploy a mobile or web application to monitor/control your device via the model, and integrate device data with business systems external to the Ayla Cloud. The video provides a high-level overview.

Devices

You can connect your device to the Ayla Cloud in three ways: Integrate it with an Ayla Production Module (circuit board) via SPI or UART; link the application that runs on it with an Ayla Software Agent; or connect it to an Ayla Gateway. For more information, see Devices.

Getting Started

Many firmware engineers start by experimenting with an Ayla Design Kit like the one pictured here. See the Production Modules guide for step-by-step instructions. Arduino boards and Raspberry Pies are useful for experimenting with Ayla agents and gateways.
